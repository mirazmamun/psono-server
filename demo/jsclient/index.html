JS Client

This is a small demo client for the password manager in JavaScript. All the tiny steps which are normally hidden from the user or done automatically in the background, are here explicitly shown as separate steps. We hope this will give a better understanding of the necessity of some steps together with an explanation of what to do in which order and why to do it.

  • Registration
  • Verify Email
  • Login
  • Logout
  • Read Datastore
  • Write Datastore

Step 1

Email and password are taken to generate the authkey for registration. The authkey is used like the password in normal registration processes as we dont want to send the password or any "easy" hashs to the server. We are using scrypt and a "special sauce" to generate this hash.

var n = 16384 // 2^14;
var r = 8;
var p = 1;
var l = 64;
var special_sauce = 'c8db7c084e181fbd0c616ed445545375a40d9a3ddc3f9d8fac1dba860579cbc1'

to_hex(scrypt(password, to_hex(sha512(lower(email)+special_sauce)), n, r, p, l))
